      このコンソールについて
      
        
      Hayabusa DFIR スキャナの localhost 用 GUI です。同梱の Hayabusa
          CLI を薄くラップした制御プレーンとして動作します。EVTX のパース
          と Sigma マッチングはすべて上流のバイナリに委譲しています。
          詳細な検知基盤の設計は ARCHITECTURE.md を参照して
          ください。
      
        
      このコンソールは単一ユーザー・localhost 限定の
          利用を前提としています。認証はありません。待ち受けポートを
          ホスト外に公開しないでください。
